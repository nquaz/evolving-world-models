# Development log — July 20, 2026

We established the first working foundation for the evolving world-models
project:

- Implemented generic named transition models for
  $p(x' \mid x, \mathrm{pa}(x))$, including immutable assignments,
  validated categorical distributions, tabular MDPs, sampling, and stable log
  probabilities.
- Implemented factored MDPs whose joint transition is the product of local
  kernels, including synchronous cross-factor parent resolution.
- Added an inherited JSON-like representation via `AbstractMDP.__str__` and
  `to_dict()`.
- Added two-slice graphical-model visualization through NetworkX and
  Matplotlib, plus optional Graphviz export. Nested factors, external parents,
  deterministic layouts, labels, domains, and styling are supported.
- Expanded `Factored_MDP_Demo.ipynb` with a weather–umbrella example that checks
  exact factor products, normalization, action effects, and graph rendering.
- Added optional `viz`, `graphviz`, `notebooks`, and `dev` dependencies to
  `pyproject.toml`, along with API and visualization documentation.
- Finished with 33 passing unit tests covering transition semantics,
  validation, representations, graph structure, rendering, and optional
  dependency behavior.

The transition-model core remains standard-library-only; visualization and
notebook tooling are opt-in extras.
